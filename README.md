# SampleExt
Sample extention for Photoshop.


## Quickstart
1. Install package.
    
        $ npm install

1. Edit Gruntfile.coffee and change 'UserName'.
    
        dest: '/Users/UserName/Library/Application Support/Adobe/CEPServiceManager4/extensions/com.example.sampleext'

1. Run
    
        $ grunt
